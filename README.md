# What your project does
Menu Decoder is a tool that allows users to have a greater undertanding of restaurants before they eat at them, with efficiency. Instead of having to manually search through menus for information, users can copy and paste website menus into this tool to recieve a summary of important information. This summary includes cuisine type, price range, average dish price, vegetarian friendly (true/false), and pescatarian friendly (true/false). Vegetarian and pescatarian information is especially useful for people who have dietary restrictions in those areas. The tool will also alert users of any extreme allergens that appear often in the menu. My project is more simple than complex, and does not dive into much beyond the MVP as described in the proposal. This is due to dealing with many parsing errors throughout, and wanting to deeply understand everything the tool includes.

# How to install and run it
1. run python menu_decoder.py
2. find online menu from a restaurant of choice (no pdfs, must be copy and pasted text)
3. paste menu under "Paste your menu text below. When finished, type END on its own line and press Enter."
4. Under pasted menu, type END on it's own line and press enter as previous instructions suggest.
